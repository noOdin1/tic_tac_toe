Instructions:

Quick start:

Press "Start Game" to start the game as "Player 1" with "X" marker and as the first player to place a marker on the board.

Starting the game:

  • Enter your name in the input box to be shown on scoreboard at the very top.
  • Click on the slide/toggle switch for "Player Play:" and "Marker" to switch between the choices.

Player wins, computer wins or draw, the player will be prompted to continue the game or not, "cancel" will end the game "ok" will continue the game. Use mouse or numeric keypad to place marker.
The first marker that gets 3 in a row wins
